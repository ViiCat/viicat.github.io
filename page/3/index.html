【Runtime】SEL、IMP、Method、Ivar、isa、Cache

Runtime 基本类型

SEL：选择器指向一个C String


IMP：指向实际执行函数体的函数指针


Method：指向方法的指针


Ivar：成员变量指针


Property：属性指针

isa：指向类的指针；id：


Cache

每当实例对象接收到消息时，它不会直接在isa指针指向的类方法列表里遍历查找能响应的方法，因为每次查找效率都太低，而优先在Cache中查找。
Runtime系统会把被调用的方法存进Cache里，就像计算机组成原理一样CUP绕过主存先访问Cache一样。
