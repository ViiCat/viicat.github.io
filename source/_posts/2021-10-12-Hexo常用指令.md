## 编译
```
$ hexo generate
```

## 本地执行
```
$ hexo serve 
```

## 发布
```
$ hexo deploy
```

## 新建文章
```
$ hexo new "article-title"
```

## 写文章
然后就可以在`source/_posts/`找到刚刚创建的`xxx.md`文件，打开xxx.md即可开始写文章了
